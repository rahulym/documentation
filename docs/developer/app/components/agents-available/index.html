app.agentsAvailable

app.agentsAvailable() is used to a list of online agents. ​

Output Parameters#

Array of objects, with all the details of the online agents. ​

Example Usage#

app.agentsAvailable().then((agents) => {
app.log(agents, "agentsAvailable")
});
