How to mask sensitive information in your bot?

In any chatbot, you can scroll up to see previous chat, but what if you chat contains sensitive information like your credit card number which you've used earlier to order something? In that case, it's good to mask sensitive information in your bot so no one can see that information.

  1. Open Studio and nevigate to a journey where you're dealing with sensitive information. Open a step in which you're asking for that sensitive information. For example, in the Verify OTP step we're asking for an OTP. In that step, open Step Setting.

  1. In the Step Setting, under mask sensitive info click to enable it for User Message. You can also enable it for Bot Message in case your bot is prividing any sensitive information.

  1. In the same step, open Channel options. Under Website tab, click on the button below sensitive information to enable it. Don't forget to click on Save button. Here we're setting it up for Website channle only but You can do the same for other channels also.

  1. Now open your bot, and try using that journey where you're dealing with sensitive information. When you're typing it, it'll be shown as dotes only. Once the message it sent, it'll be shown as Sensitive information.
