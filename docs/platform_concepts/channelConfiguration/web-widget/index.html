Change the name,icon, description of the bot?#

Go to Configure tab from the left sidebar. Here you can change the Name, Industry, Description as well as Icon of the bot.

How to change the font of a bot?#

Good fonts means 5 more marks for good handwriting! Haha! Let's see how we can change the font of the bot. This document will help in changing font for PWA/ Website Bots.

Keywords : #fonts, #chatWidget #botUI References : Google Fonts

Step 1 : Browse through the Google fontslink and choose your favorite style.

Step 2 : Select a style, get the URL. URL would look something like this - Google Fonts

Step 3 : Open this link in a different tab and get the .woff2 link from there.

Step 4 : Now add this link to the Channels > Web > Font > Advanced > Custom Font URL
